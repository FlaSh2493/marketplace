# Worktree Status

**실행 주체: Main Session 전용**

## 사용법
`/worktree-flow:status`

## 실행 절차

STEP 1: 워크트리 목록 조회
  실행: `git worktree list --porcelain`
  메인 리포(첫 번째 항목) 제외한 워크트리 목록 파싱

  각 워크트리마다:
    - 브랜치: `git -C {경로} rev-parse --abbrev-ref HEAD`
    - WIP 커밋 수: `git -C {경로} log --oneline | grep -ic "wip("`
    - 마지막 커밋: `git -C {경로} log -1 --format="%ci"`

STEP 2: 결과 출력
  ```
  ┌──────────┬──────────────────────────┬────────┬────────────────────────┐
  │ 이슈     │ 브랜치                    │ WIP수  │ 마지막 커밋             │
  ├──────────┼──────────────────────────┼────────┼────────────────────────┤
  │ PLAT-101 │ worktree-PLAT-101        │  3개   │ 2026-03-25 14:32:01    │
  └──────────┴──────────────────────────┴────────┴────────────────────────┘
  ```
  워크트리가 없으면: "활성 워크트리 없음"
