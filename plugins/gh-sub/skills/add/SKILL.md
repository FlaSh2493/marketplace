---
name: gh-sub-add
description: Add a new GitHub account profile with name, email, and SSH host alias.
---

To add a new GitHub profile, I need the following information:
1. **Profile ID**: A short name for the account (e.g., 'main', 'work', 'personal').
2. **User Name**: The name used for git commits.
3. **User Email**: The email address used for git commits.
4. **SSH Host Alias**: The alias defined in your `~/.ssh/config` (e.g., `github.com-sub`).

I will then save this profile to `~/.gh-sub/profiles.json`.
