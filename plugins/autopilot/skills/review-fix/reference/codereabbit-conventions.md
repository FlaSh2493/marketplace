# CodeRabbit Conventions

- **Bot Account**: `coderabbitai[bot]`
- **Resolved**: Indicated by a `+1` reaction on the root comment of a thread.
- **Replies**: CodeRabbit may reply to its own comments. All replies in a thread should be aggregated for context.
- **Outdated**: A comment is outdated if the file path no longer exists in the current worktree or the line has been significantly changed (though we rely on `git ls-files` and GitHub's `outdated` flag).
