---

사용자가 프로젝트 키 또는 이슈 목록 조회를 요청하면 아래 명령을 실행한다.

```
cd <plugin-root>/scripts && python list.py <ARGS>
```

## 인자 규칙

| 입력 예 | 설명 |
|--------|------|
| `MKT` | 단일 프로젝트 (활성 스프린트 × 본인 할당) |
| `MKT,ADX` | 다중 프로젝트 (콤마) |
| `MKT ADX` | 다중 프로젝트 (공백) |
| `MKT --all` | 스프린트 전체 (할당 제약 해제) |
| `MKT --jql "priority = High"` | 추가 필터 |

## 결과 처리

- 출력된 `KEY  [Status]  Title` 목록을 그대로 사용자에게 보여준다.
- 파일을 저장하거나 읽지 않는다.
- 이슈 내용을 fetch하려면 `/jsync:fetch <KEY>`를 사용한다.
